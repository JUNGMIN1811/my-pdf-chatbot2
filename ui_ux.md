노무(노동 관련) 챗봇 웹애플리케이션을 만들어줘. 요구사항:

1. 기술 스택
- React + TypeScript로 구축
- 스타일링은 Tailwind CSS 사용
- 대화형 UI (채팅 인터페이스)

2. 색상 스킴 (네이버 스타일)
- 주요 색상: 초록색 (#00C73C 또는 #1EC800)
- 보조 색상: 진회색 (#333333), 연회색 (#F5F5F5)
- 텍스트: 검정색 (#000000), 연회색 (#999999)
- 하이라이트: 파랑색 (#0066FF)

3. 기능
- 상단 헤더: 로고 + "노무 챗봇" 제목
- 채팅 영역: 사용자 메시지(오른쪽, 파랑배경)와 봇 응답(왼쪽, 회색배경)
- 입력창: 텍스트 입력 + 전송 버튼(초록색)
- 샘플 질문: "최저임금 조회", "퇴직금 계산", "휴가 일수", "4대 보험" 등

4. 레이아웃
- 반응형 디자인 (모바일 최적화)
- 최소 폭 400px, 최대 폭 900px
- 채팅창 높이: 400-500px (스크롤 가능)

5. 사용자 경험
- 초기 상태: 인사말 + 카테고리별 버튼
- 메시지 입력 시 실시간 반응
- 봇의 모의 응답 (실제 API 연결 불필요)
